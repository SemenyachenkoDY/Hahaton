# План выполнения (Roadmap) - Milestone 2: Публикация Репозитория

## Этап 7: Подготовка локального Git
- Инициализация локального репозитория, добавление `.gitignore` для фронтенда и пайтона.
- Первичная фиксация (Initial commit) всех наших файлов Майлстоуна 1.

## Этап 8: Подключение и Push на GitHub
**Depends on:** 7
- Подключение к вашему пустому удаленному репозиторию на GitHub (потребуется URL).
- Отправка файлов в ветку `main`.
- После отправки, GitHub Actions (`deploy.yml`) автоматически развернет Frontend на GitHub Pages!
