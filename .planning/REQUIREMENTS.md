# Требования

## 1. Спецификация Функционала
- **Главная Страница**: Яркий красивый лендинг хакатона в "огненном" теплом стиле.
- **Оболчка Дашборда**: Гибкий layout с явными контейнерами-местами для подстановки графиков и визуализаций данных.
- **Интеграция API**: Модульная архитектура во `frontend` для быстрого подключения API от сторонних сервисов или `backend`. Возможные конфиги (env переменные) для сторонних сервисов.
- **Совместимость с GitHub Pages**: Сборка во фронтенд статике (строгий роутинг программно или правильный `base` путь Vite).
- **Спецификация Бэкенда**: Готовый файл `docker-compose.yml`, описывающий какие бэкенд контейнеры мы подразумеваем в проекте (БД, Python-API).

## 2. Эстетические Требования
- Цвета: Переход от белого к сочному оранжево-огненному градиенту по мотивам зажженной спички (премиум-вид).
- Современные, чистые контейнеры-карточки со стеклянным (glassmorphism) или минималистичным эффектом.

## 3. Деплой и Инфраструктура
- Архитектура строго разбита на папки: `/frontend` и `/backend`.
- Фронтенд собирается на GitHub Pages.
- Бэкенд и аналитика разворачиваются через Docker Compose локально/виртуально.
